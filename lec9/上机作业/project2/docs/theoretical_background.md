# 理论背景

本项目围绕糖尿病数据集对比三种典型的线性回归方法：普通最小二乘（OLS）、岭回归（Ridge）与 Lasso。三者共享线性假设 $\hat{y} = X\beta + b$，差异体现在目标函数中的正则化项。

## 普通线性回归

- 目标函数：$\displaystyle \min_{\beta} \|y - X\beta\|_2^2$
- 解析解：$\hat{\beta} = (X^\top X)^{-1} X^\top y$
- 特点：无正则项，容易过拟合，且对多重共线性敏感。

## 岭回归

- 目标函数：$\displaystyle \min_{\beta} \|y - X\beta\|_2^2 + \lambda \|\beta\|_2^2$
- 解析解：$\hat{\beta} = (X^\top X + \lambda I)^{-1} X^\top y$
- 特点：通过 $L_2$ 惩罚限制系数幅度，缓解多重共线性，但不会产生真正的稀疏解。

## Lasso 回归

- 目标函数：$\displaystyle \min_{\beta} \|y - X\beta\|_2^2 + \lambda \|\beta\|_1$
- 特点：$L_1$ 正则化可促使部分系数精确为零，实现特征选择与模型压缩。

## 正则化路径

改变 $\lambda$ 会形成“系数路径”：
- 较小 $\lambda$ 时，模型接近 OLS，偏差低但方差高；
- 较大 $\lambda$ 时，系数收缩，偏差升高但方差下降。

岭回归路径是平滑的；Lasso 路径呈折线状，并在某些点将特征系数压为零。通过绘制路径可以直观对比两种正则化策略的影响。
